#  FLASHBACK

* Unidad: *Hackathon*
***
El presente proyecto consiste en crear un producto digital con la temática de películas.

## Herramientas
***

* Materialize (Framework).  
* Jquery (Librería JS).  
* Firebase (Plataforma de desarrollo).  
* OMDb API.
